# Final-Project
Final Project for Data Science Programming.

This project uses a dataset containing all EV and Hybrid Vehicle information in the state of Washington. Using this data, this project analyzes and answers the following questions:

 1. Which car brands have the biggest market share in high-population counties compared to rural ones?
 2. Based on range, is the drivable range increasing each model year?
 3. What is the market split between pure electric cars and hybrid vehicles?
 4. How old is the average EV by make as compared the average Hybrid by make?
