# Final-Project
Final Project for Data Science Programming.

This project uses a dataset containing all EV and Hybrid Vehicle information in the state of Washington. Using this data, this project analyzes and answers the following questions:

 1. Which car brands have the biggest market share in high-population counties compared to rural ones?
 2. Based on range, is the drivable range increasing each model year?
 3. What is the market split between pure electric cars and hybrid vehicles?
 4. How old is the average EV by make as compared the average Hybrid by make?

The following files are included in this project:
compressed_data.csv.gz - Full Dataset (had to compress due to immense file size).
Cleaned_EV_Data - Cleaned dataset. Contains only the rows/columns that are required to answer questions.
CleaningEV - Python Notebook used to clean dataset. 
Final Analysis - Python Notebook used to analyze cleaned dataset and answer questions/make plots and graphs.
One Pager - A one page summary of this project.
Present - Slides Presentation summarizing the findings of the project and the methods used to obtain them.
